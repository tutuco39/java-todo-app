# CLI Todo アプリ

Java で作成した、シンプルな CLI ベースの Todo 管理アプリです。

オブジェクト指向の基礎（責務分離・状態管理）を学ぶ目的で作成しました。

## 何ができるか（機能）
- タスクの追加
- タスクの一覧表示
- タスクの完了切替
- タスクの削除

## どうやって動かすか（実行方法）

```bash
javac -d bin *.java
java -cp bin Main

```
```md
## 使い方

起動後、以下のメニューが表示されます。

1: 追加  
2: 一覧  
3: 完了切替  
4: 削除  
0: 終了  

番号を入力して操作します。
```

## 設計の考え方（クラス構成・責務）

- Main  
  アプリのエントリーポイント。  
  入力を受け取り、処理を各クラスに委譲する。

- Task  
  タスクを表すエンティティ。  
  タイトルと完了状態を持つ。

- TaskManager  
  タスク一覧を管理するクラス。  
  追加・削除・完了切替などのロジックを担当。

- CliView  
  表示専用クラス。  
  タスク一覧やメッセージの表示を担当。
